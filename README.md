# carusel_bulder

Adobe After Effects ExtendScript tools for carousel rigs.

## Current Working Scripts

- `photo/carusel_builder_RV_photo_v8_3.jsx` - photo/video carousel script, based on the stable v8.2 branch.
- `text/carusel_builder_RV_text_v8_3.jsx` - separate text-block carousel script, cleaned up so it creates only the requested number of text blocks.

## After Effects

Tested locally in Adobe After Effects 2022.

## Notes

This repository keeps the working scripts available from another computer. The scripts are intentionally split into photo and text versions so each rig stays clean and easier to maintain.
